There are plenty of websites and tools out there to create your own iceberg tier lists, but none of them seem to go beyond adding words to a PNG and calling it a day. So, I've spent a good few weeks [vibe] coding a *ridiculously* feature-packed tool that allows you to do so much more!

It started out with the simple idea to just add descriptions to entries, but it's grown to become a lot more as I've worked away at it.

[**Link is here, have fun!**](https://eight8100.github.io/interactive-iceberg/)

Mobile support is included, but less polished/usable than desktop - this will be worked on!

## Features

- Drag & drop entries anywhere on the iceberg, and organise by tier
- Rich descriptions, images, even YouTube embeds!
- Ability to link related entries with each other via the pickwhip tool (⊕)
- Pin favourites (★) to a dedicated panel for quick access
- You can search titles and descriptions; or find a random entry with the 🎲 icon
- Customisable tier names with image support
____
- Display controls for text size, background blur, font, tier titles, and water effect
- Lock the iceberg to prevent accidental edits while viewing
- Highlight multiple entries to move them around en masse
____
- Browser autosave plus local .zip save/load, with a restore prompt on startup
- Export your iceberg as a PNG for easy sharing

  Entry images do **not** save to the broswer's cache due to a 5MB limit (on Arc anyway), but tier images do. Entry images only save to the .zip.

## Disclaimer

Yes, this is essentially vibe coded. However, I hate sloppily put together apps and sites, so I've made sure to get Claude to periodically read over the code and do extensive cleanups, to avoid declarations and classes being piled up and creating massive amounts of redundancies. The codebase has been through several in-depth cleanup passes to make sure there's little to no redundant code. Ideally none lol. I am not a web developer, nor do I have nearly enough time to even begin to learn half of what's included in this project. I've used AI as a tool, but certainly not a sloppy crutch.
